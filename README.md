# Playground to explore and experiment with libuv

Tested only on macOS.

## Pre-requisites

- This project uses CMake to build the experiments.
- libuv header and library files are needed.

```
brew install libuv
```

## Usage

- Generate build files using CMake

```
cmake -B build
```

- Build

```
cmake --build build
```

- Run
  The following is an example and there are other experiments that can be explored.

```
./build/server
```

On another terminal

```
./build/client
```
